# Сайты с генерацией тестовых данных

1. https://www.random1.ru/ Программа генерирует случайное имя, фамилию, отчество, контактные и другие данные для дальнейшей работы с ними. паспортные данные не генерирует
2. https://randomdatatools.ru/ Программа генерирует случайное имя, фамилию, отчество, контактные, паспортные и другие данные для дальнейшей работы с ними
